# Struggles & Further-Study Tracker

Concepts or skills that need extra attention — richer than the review queue,
which only tracks quiz misses. Claude adds entries when it OBSERVES difficulty
(repeated confusion during builds, needing the full hint ladder, avoidance of a
topic), not just when a quiz is failed. Claude also records resolutions, so this
doubles as a "look how far you've come" record.

Format:
- **[topic]** — first noticed: DATE (Session X.Y) | signal: what Claude observed
  | status: open / remediation scheduled / resolved DATE | plan: ...
