# Curriculum Adaptations Log

Every change made to the standard curriculum — insertions, compressions,
extensions, reorderings — with the evidence that justified it. Maintained by the
curriculum-adapter skill. Anything marked "parent review" should be discussed
with the supervising adult before taking effect.

Format:
## YYYY-MM-DD — <change>
- **Type:** insert remedial / compress / extend / reorder / swap project
- **Evidence:** quiz scores, mastery map, struggles entries, pace
- **Change:** exactly what the revised plan is
- **Parent review needed:** yes/no
